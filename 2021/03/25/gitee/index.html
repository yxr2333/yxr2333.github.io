Git的用法总结

总结一下Git的一些用法，不然时间长了忘了每次都要重新查博客

Git一般都使用命令行进行操作0.0

一、Git bash

右键文件夹,选择Git Bash here ( 命 令 行 模 式)

二、Git init 初始化

​ 将文件夹初始化为Git管理的文件夹

三 、Git add

​ git add ‘files name’ —-> 将文件名（文件夹名）为files name的文件添加到本地缓存区

四、Git commit

​ (1) commit命令是将已经存储到本地缓存区的文件，存到本地的仓库中。

​ (2) message 是上传的时候需要备注的信息

执行完上述命令后，需要被上传的文件已经存储到了我们的本地仓库中，下一步我们需要做的就是将该文件推送到远程仓库中（一般是Github仓库或者Gitee仓库）

五、Git push

​ 首先我们需要连接我们的远程仓库，而连接远程仓库的方式大致分为两种。https公钥和ssh密钥
