会遇到的问题：

1、如果需要上传大文件怎么办？

​ 解决方法：上传的文件如果大于100M那么就需要下载git lts 软件来上传

2、如何查看缓存区中有哪些文件？

​ 解决方法：输入命令 git cherry -v 即可查询到

3、如果add后想取消上传该如何处理？

​ 解决方法：输入命令 git reset HEAD~，就可以取消上传了。如果想再次上传，只需要再执行一次commit操作就可以了

4、常用的指令

​ git init 初始化

​ git add 添加进暂存区

​ git commit 上传到本地仓库

​ git push -u orgin master 上传到github仓库

​ git cherry -v 查看本地仓库中暂时还未上传的
